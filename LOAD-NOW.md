## Quick Load Steps

### Option 1: Chrome
1. **Open Chrome**
2. **Type in address bar:** `chrome://extensions/`
3. **Enable Developer mode** (toggle in top right)
4. **Click "Load unpacked"**
5. **Paste this path:**
   ```
   C:\Users\shiva\Downloads\fantastic_four-ParthVarekar-blindness-part\fantastic_four-ParthVarekar-blindness-part
   ```
6. **Click "Select Folder"**

### Option 2: Edge
1. **Open Edge**
2. **Type in address bar:** `edge://extensions/`
3. **Enable Developer mode** (toggle in bottom left)
4. **Click "Load unpacked"**
5. **Paste the same path above**
6. **Click "Select Folder"**

## After Loading

1. Click the extension icon in your browser toolbar
2. Allow microphone permission
3. Verify settings:
   - Ollama URL: `http://localhost:11434`
   - Model Name: `tinyllama`
4. Say **"Hello Vision"** to activate!
